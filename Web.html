O que são Clients?

Clients são dispositivos ou programas que solicitam serviços de servidores. Exemplos incluem navegadores web e aplicativos móveis.

Importante: Eles dependem dos servidores para obter dados.

Veja mais em MDN Web Docs.

O que são Servers?

Servers são computadores que fornecem serviços para clients, como hospedagem de sites e armazenamento de dados.

Exemplo: Um servidor web responde a requisições HTTP.

Mais informações em Servidor (computador) - Wikipédia.

Como é uma estrutura básica de um HTML?

Um documento HTML básico começa com <html> a tag raiz, seguida por <head> e <body>.

Dentro do head, definimos o título e metadados; no body, o conteúdo visível.

Exemplo: Texto com a tag <font> para mudar fonte e cor.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenadas (<ol>) exibem itens numerados, enquanto listas não ordenadas (<ul>) exibem marcadores.

Exemplo de lista ordenada:

  1. Primeiro item
  2. Segundo item

Exemplo de lista não ordenada:

  • Item A
  • Item B
